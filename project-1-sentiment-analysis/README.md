# Project 1: Sentiment Annotation (Product Reviews)

## Overview
This project focuses on labeling product review text into Positive and Negative sentiment categories. The goal is to build a consistent, high-quality dataset suitable for training machine learning models.

## Task
Each text sample is classified based on customer sentiment:
- Positive: satisfaction, approval, or meeting expectations
- Negative: dissatisfaction, complaints, or unmet expectations

## Tools Used
- Label Studio
- GitHub

## Dataset Type
Text classification (binary sentiment analysis)

## Objective
To demonstrate consistent annotation skills and ability to follow labeling guidelines for AI training data.

## Status
In progress – dataset creation and annotation ongoing.
