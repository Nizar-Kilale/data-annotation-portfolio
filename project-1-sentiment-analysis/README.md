# Project 1: Sentiment Annotation (Product Reviews)

## Overview
This project involves annotating product review text into Positive and Negative sentiment categories. The objective is to create a clean, consistent dataset suitable for machine learning applications.

---

## Dataset Versions
- **dataset-v1.csv** → Initial export from Label Studio (raw format)
- **dataset-clean-v1.csv** → Cleaned and structured dataset for ML use

---

## Annotation Guidelines
Each review is labeled based on customer sentiment:

### Positive
- Satisfaction is expressed  
- Product meets expectations  
- User is happy or satisfied  
- Recommendation is implied  

### Negative
- Dissatisfaction is expressed  
- Expectations are not met  
- Complaint or regret appears  
- Frustration or disappointment exists  

### Key Rule
If unclear, label based on whether the sentence leans toward satisfaction or dissatisfaction.

---

## Workflow
1. Data imported into Label Studio  
2. Text annotated using defined guidelines  
3. Dataset exported in CSV format  
4. Raw dataset cleaned and standardized  
5. Final dataset prepared for ML usage
